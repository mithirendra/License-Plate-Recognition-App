# License Plate Recognition (LPR) App
A Python-based OCR app that detects and reads Malaysian license plates from uploaded images.

## Features
- Upload an image and extract license plate text
- Built using EasyOCR for optical character recognition
- Supports Malaysian license plate formats
- Simple web interface built with Streamlit

## Built With
- Python
- EasyOCR
- Streamlit
- OpenCV

## Current Accuracy
Approximately 65% — works best on clear, well-lit images with standard plate fonts.
